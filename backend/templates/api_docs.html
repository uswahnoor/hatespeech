POST /api/detect/

Detect hate speech and toxicity in text content

Headers:

Content-Type: application/json
X-API-KEY: your-api-key-here

Request Body:

{
  "text": "Text to analyze for hate speech"
}

Response:

{
  "classification": "safe",
  "confidence": 0.95,
  "sentiment": "positive",
  "engine": "transformer",
  "latency_ms": 145.2
}

Get API documentation in JSON format

üìã Response Fields

  • classification: Either "safe" or "toxic"
  • confidence: Confidence score between 0 and 1
  • sentiment: "positive", "negative", or "neutral"
  • engine: AI model used for detection
  • latency_ms: Processing time in milliseconds

üîß Example Usage

JavaScript (fetch)

const response = await fetch('http://localhost:8000/api/detect/', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'X-API-KEY': 'your-api-key-here'
  },
  body: JSON.stringify({
    text: 'Hello, how are you today?'
  })
});

const result = await response.json();
console.log(result.classification); // "safe" or "toxic"

Python (requests)

import requests

response = requests.post('http://localhost:8000/api/detect/', 
  headers={'X-API-KEY': 'your-api-key-here'},
  json={'text': 'Hello, how are you today?'}
)

result = response.json()
print(result['classification'])  # "safe" or "toxic"

cURL

curl -X POST http://localhost:8000/api/detect/ \
  -H "Content-Type: application/json" \
  -H "X-API-KEY: your-api-key-here" \
  -d '{"text": "Hello, how are you today?"}'
